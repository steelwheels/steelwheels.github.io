Steel Wheels Project Web Page

Introduction

Welcome to Steel Wheels Project web page. This project is started to develop open source software for macOS X or iOS (iPhone family devices). You can get applications, frameworks and documents from this project.

Framework, Library

This is the list of frameworks/libraries produced by this project. They are targetting macOS and iOS and written by Swift language.

Frameworks, Libraries
Name Description
Canary Framework Package of general purpose data structure and classes.
KiwiGraphics Framework Package of data structure and classes for graphics.
KiwiControls Framework Package of GUI controls which can be used by macOS and iOS.
KiwiComponents Framework Package of GUI components based on MdVsC design pattern.
